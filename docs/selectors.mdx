Selectors are used apply checks constraints and constraints to subsets of packages in a project and are defined in a [project configuration file](/docs/api/configuration#project-configuration).

## Select all packages

Use the `*` selector to apply constraints and checks to all packages.

**Example check:**

```ts commonality.config.ts
import { defineConfig } from 'commonality';

export default defineConfig({
  checks: {
    '*': [recommended.hasReadme()],
  }
});
```

**Example constraint:**

```ts commonality.config.ts
import { defineConfig } from 'commonality';

export default defineConfig({
  constraints: {
    '*': { disallow: ['deployable'] },
  },
});
```

## Select packages with a tag

Select a tag to apply constraints and checks to a subset of packages.

**Example check:**

```ts commonality.config.ts
import { defineConfig } from 'commonality';

export default defineConfig({
  checks: {
    'buildable': [
      recommended.hasJsonFile('package.json', {
        scripts: {
          build: 'tsc --build',
          dev: 'tsc --watch'
        }
      })
    ],
  }
});
```

**Example constraint:**

```ts commonality.config.ts
import { defineConfig } from 'commonality';

export default defineConfig({
  constraints: {
    config: { disallow: ['application'] },
  },
});
```
