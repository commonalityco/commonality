---
title: defineCheck
---

```ts
(checkCreator: () => Check) => () => Check
```
This utility provides type-safety and intellisense for check options.
This utility is a higher-order function that encourages defining checks as functions to make them composable.

## Parameters
<ParamField path="checkCreator" type="() => Check" required>
  A function that returns a [Check](/sdk/check-object) object.
</ParamField>

## Returns

Returns the original that was passed in.

## Example

```ts
import { defineCheck } from 'commonality';

const hasCodeowner = defineCheck(() => {
  return {
    // Check object
  };
});
```
